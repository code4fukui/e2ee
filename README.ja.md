# e2ee

エンドツーエンド暗号化 (E2EE) メッセージングを支援するブラウザベースのツールです。鍵交換にはEd25519/X25519、暗号化にはAES-256-GCMを使用します。

## デモ

[E2EE - Ed25519 version](https://code4fukui.github.io/e2ee/)

## 特徴

-   **セキュアな暗号化**: キーペア生成にはEd25519、共有鍵の導出にはX25519、メッセージの暗号化/復号にはAES-256-GCMを実装しています。
-   **ブラウザベース**: Web Crypto APIを使用してブラウザ内で完結して動作します。
-   **サーバーレス動作**: ページの初回読み込み後はサーバーとの通信を行わず、プライバシーを確保します。
-   **自己完結型**: インストールすべき外部依存関係のない、単一のHTMLファイルです。

## 使い方

デモでは、エンドツーエンド暗号化を実行するためのシンプルなインターフェースを提供しています。

1.  **キーペアの生成**: **"1. キーペア（公開鍵と秘密鍵）をつくる"** ボタンをクリックして、秘密鍵と公開鍵を生成します。秘密鍵は誰にも見せないでください。
2.  **公開鍵の交換**: 生成した **公開鍵** を通信相手と共有します。相手から受け取った公開鍵を **"2. 相手の公開鍵..."** フィールドに貼り付けてください。
3.  **共有鍵の作成**: **"3. 通信用の共通鍵をつくる"** ボタンをクリックします。これにより、あなたの秘密鍵と相手の公開鍵を使用して、お互いに同一となる共通鍵が導出されます。
4.  **暗号化**: メッセージを左のテキストエリアに入力し、**"暗号化 →"** をクリックします。暗号化された暗号文が右に表示されます。
5.  **復号**: メッセージを復号するには、暗号文を右のテキストエリアに貼り付け、**"← 復号"** をクリックします。元のメッセージが左に表示されます。
